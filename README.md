A simple web-based password generator built using Python and Gradio. This application generates strong and secure passwords based on the length provided by the user.

🚀 Features

Generate random secure passwords

Custom password length input

Uses letters, numbers, and special characters

Handles invalid input errors

🛠 Technologies Used

Python

Gradio

Random & String modules

▶️ How to Run

Install the required library:

pip install gradio

Run the Python script or notebook.

A local web interface will open where you can enter the password length and generate a secure password.

📂 Project File

password_generator.py or password_generator.ipynb – Main program file
